MySQLfront训练题目2

以下的题目均出至于5月6日的课堂练习题请大家认真对待每一题,时间关系后面我会进行各文章的项目分类目

1.写出学生和班级具有外键完整性约束的SQL语句,其中
班级表class:主键id、班级名称name
学生表student:主键id、姓名name、年龄age、班级号cid(外键、引用班级表的主键id)

2.使用连接查询方式,写出查看所有学生所在班级信息(有学生还未分配班级)的SQL语句,输出字段为学生姓名、班级名称。

3.使用子查询方式,写出查询所在班级名称不是“2001班”的学生信息,输出字段为学生姓名、学生年龄

答案：

1.create table class(
id int primary key,
name varchar(50)
);
create table student(
id int primary key,
name varchar(50),
age int,
cid int,
foreign key(cid) references class(id)
);

2.select a.name,b.name from student a left join class b on a.cid = b.id

3.select name,age from student
where cid not in (select id from class where name = ‘2001班’)

------ END ------
