MySQLfront训练题目

以下的题目均出至于3月29日的课堂练习题请大家认真对待每一题,时间关系后面我会进行各文章的项目分类目

一.简答题
1.新增数据库 DB_2023
2. 选择刚新建的数据库 DB_2023

答案：
create database DB_2023;
use DB_2023;

1.主键 id 要求为主键和自动增长,
2.姓名 name 要求不能为空,
3.学号 no 要求不能重复,
4.性别 sex ,
5.成绩 score，

答案：
create table student(
id int primary key auto_increment,
name varchar(32) not null,
no varchar(32) unique,
sex varchar(32),
score int default 0
);

1.修改表的名字为 stu
2.修改学号 no 字段的顺序调到主键 id 后面
3.修改性别 sex 的字段名称为 gender,类型为 char(50)
4.给表新增 class 字段,类型为 varchar(30)
5.给 score 字段创建索引,索引名称为 index_score

答案：
alter table student rename stu;
alter table stu modify no varchar(32) after id;
alter table stu change sex gender char(50);
alter table stu add class varchar(30);
或 alter table stu add index index_score(score);

1.新增一条记录：id 为 5,学号为 1001,姓名为张一,性别为男,成绩为 99,班级为 2023班
2.新增一条记录：学号为 1002,姓名为张二
3.批量新增三条记录：学号为 1003,姓名为张三;学号为 1004,姓名为张四;学号为 1005, 姓名为张五
4.更新学号为 1002 的学生信息：性别为女,班级为 2023 班,成绩为 90
5.删除学号为 1005 的学生信息

insert into stu values(5,’1001’,’张一’,’男’,99,’2023 班’);
insert into stu(no,name) values(‘1002’,’张二’);或 insert into stu set no=’1002’,name=’张二’;
insert into stu(no,name) values(‘1003’,’张三’), (‘1004’,’张四’), (‘1005’,’张五’);
update stu set gender=’女’,class=’2023’,score=90 where no=’1002’;
delete from stu where no=’1005’;

1.清空 stu 表的所有数据
2.删除表 stu
3.删除数据库 DB_2023

delete from stu;或 truncate stu;
drop table stu;
drop database DB_2023;

这就是全部的练习了希望大家能好好学习一起进步
